# Changelog

All notable changes to Gold Mine will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-09

### Added
- Initial Unity project setup
- Core GameManager singleton system
- GameState management for match data
- CardData system for card definitions
- 8-bit retro visual aesthetic
- GitHub repository with documentation
- Project structure and folder organization
- .gitignore for Unity
- Contributing guidelines
- README with game overview

### Planned
- MatchController for game loop
- Unit and Tower behavior systems
- AI Controller for enemy logic
- Main menu UI
- In-game HUD
- Shop system UI
- Sprite assets
- Audio system
- Mobile optimization

## Versioning

- **Major.Minor.Patch**
- Increment MAJOR for incompatible API changes
- Increment MINOR for backward-compatible additions
- Increment PATCH for bug fixes

## Release Process

1. Update CHANGELOG.md
2. Update version in code
3. Create git tag: `v0.1.0`
4. Push to GitHub
5. Create Release on GitHub
